The GR8 CRM Web

The GR8 CRM plugin crm-content can be used to serve HTML pages for a web site.

Just create pages using the Rich Text Editor included in the crm-content-ui plugin. Then set the access rights on the pages to public and your site is ready to use. You can import static HTML pages from src/templates/crm and initialize your new site in seconds.

The only thing you need to add to your empty Grails application is:

  1. Three GR8 CRM plugins in BuildConfig.groovy
  2. One line in UrlMappings.groovy
  3. One Groovy Server Page (.gsp) template
  4. One custom layout (if you're not satisfied with the default)
  5. A few lines of initialization code in BootStrap.groovy

Clone the github repository and see for yourself!
